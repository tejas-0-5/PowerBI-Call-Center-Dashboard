# Power BI Call Center Dashboard

This project is an interactive Power BI dashboard built to analyze call center performance using KPIs and visual analytics.

---

## Project Overview

- Tool: Microsoft Power BI
- Domain: Call Center Analytics
- Type: End-to-End Dashboard Project
- Data Processing: Power Query
- Calculations: DAX

---

## Objectives

- Track total calls over time
- Analyze call duration and response efficiency
- Understand customer sentiment
- Identify top reasons for calls
- Compare performance across call centers and regions

---

## Key KPIs

- Total Number of Calls
- Total Call Duration (Hours and Minutes)
- Average Call Duration
- Response Time Percentage
- Call Sentiment Distribution

---

## Visualizations

- Column Chart – Total Calls by Day
- Filled Map – Total Calls by State
- Tree Map – Top Reasons for Calls
- Donut Chart – Calls by Channel
- Bar Chart – Calls by Call Center
- KPI Cards – Performance Summary
